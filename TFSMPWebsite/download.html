Player

The Player is the mod that provides access to Turboprop Flight Simulator Multiplayer. Download and install it now to play with others over the Internet!

The player is only supported on Android 4.4 and above.

Server

The server software allows you to host your own TFSMP servers for people to connect to. You can customize your server by changing its settings and installing new plugins.

The server requires Python 3.10 installed on your device to work.

Read our server guide - it's a comprehensive guide for setting up your server.
